¿Qué dirección visual prefieres para NexusEdu?

El diseño debe sentirse como un juego educativo profesional. Considera cómo quieres que se sienta la experiencia día a día.

⚡
Neon Cyberpunk

Cyberpunk / Neon

Colores vibrantes (magenta, cyan, púrpura), bordes afilados, UI futurista. Energía alta, edgy, muy gamer. Se siente como un FPS o batalla táctica.

🎮
Gradientes Modernos

Gradientes Suaves

Gradientes sutiles (purpura → azul), bordes redondeados, glassmorphism. Profesional pero divertido. Equilibrio entre juego y educación.

█
Brutalist

Brutalist / Retro

Monocromo con accent color (neon verde), bordes duros, tipografía monoespaciada. Vibe hacker/terminal, minimalista extremo.

🌟
Soft & Playful

Soft & Playful

Colores cálidos (naranja, rosa), bordes muy redondeados, animaciones bounce. Accesible, amigable, menos "hardcore gaming".

Nota: Estos son los tonos generales. El sistema actual usa naranja (#ff6b00) como primario. Podemos mantenerlo, evolucionarlo, o cambiar completamente según tu preferencia.
