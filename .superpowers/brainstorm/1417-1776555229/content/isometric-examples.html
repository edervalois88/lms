Pixel art isométrico 3D personalizable

Con outfit, mascota y accesorios

Clásico 32x32

Estilo retro auténtico, muy detallado. Cada cosmetic añade capas visibles.

  • ✓ Muy personalizable (muchas combinaciones)
  • ✓ Nostálgico y único
  • ✓ Excelente para mecánicas de gamification
  • ✗ Requiere muchos assets (~200+ sprites)

Limpio y minimalista

Moderno 64x64

Más limpio y contemporáneo. Mejor para interfaces minimalistas.

  • ✓ Simple de crear y mantener
  • ✓ Carga muy rápido
  • ✓ Se ve bien en cualquier tamaño
  • ✗ Menos espacio para detalles

Con animación y cosmetics superpuestos

Híbrido SVG (Recomendado)

Lo mejor de ambos mundos: pixel art + flexibilidad técnica.

  • ✓ Completamente escalable
  • ✓ Fácil de animar (Vue + SVG)
  • ✓ Cosmetics como componentes Vue
  • ✓ Reutilizable en cualquier tamaño
  • ✓ Mejor para educación (clean & profesional)
