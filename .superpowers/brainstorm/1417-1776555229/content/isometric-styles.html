Estilos de Pixel Art Isométrico para Avatares

¿Cuál estilo prefieres para NexusEdu?

🧑‍🎓

Clásico 32x32

Estilo retro auténtico, muy detallado. Como viejos videojuegos RPG.

  • ✓ Muy personalizable
  • ✓ Nostálgico
  • ✗ Requiere muchos assets
🎓

Moderno 64x64

Más limpio, menos detalle. Mejor para interfaces limpias.

  • ✓ Simple y limpio
  • ✓ Carga rápida
  • ✗ Menos expresión
👨‍🎓

Híbrido SVG

Pixel art renderizado con SVG. Escalable y animable fácilmente.

  • ✓ Escalable
  • ✓ Animable
  • ✓ Flexible con cosmetics

Técnica de Composición (Todas las opciones)

Independiente del estilo, los avatares se compondría en capas:

🟫 Base (cuerpo isométrico)
+ 🟪 Color (skin tone)
+ 👕 Outfit (ropa)
+ 🎒 Accessory 1 (mochila, hat, etc)
+ ✨ Accessory 2 (gafas, etc)
+ 🐉 Pet (mascota)
= 👨‍🎓 Avatar Final
